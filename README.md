# Task Manager API 🚀

API REST para gestión de tareas desarrollada con Node.js y Express.

## Funcionalidades
- Crear tareas
- Listar tareas
- Eliminar tareas
- Estructura modular (MVC)

## Tecnologías
- Node.js
- Express

## Endpoints

GET /api/tasks  
POST /api/tasks  
DELETE /api/tasks/:id  

## Ejecución

```bash
npm install
npx nodemon src/app.js
